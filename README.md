# Automated Sprinkler System
Hi! this was made by the researchers from their paper "***The Use of DIY Automated Sprinkler System on Indoor Plants***".

# Materials Used

 - [ ] Arduino Uno R3
 - [ ] Wires
 - [ ] 2.5L Water Container
 - [ ] Wood(for frame)
 - [ ] Seeds
 - [ ] RTC Module(DS1302)
 - [ ] LCD I2C
 - [ ] 12V DC Motor
 - [ ] Ultra Sonic Sensor
 - [ ] Toshiba TTC5200 NPN Transistor
